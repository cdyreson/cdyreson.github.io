Marker

Marker marks a quiz, generates statistics on each question, and emails a complete response to students (and to you if you so desire). The key feature of Marker is that it tells the user exactly which questions were missed and why.

Marker is not a single program, rather it is several programs.

  1. grader - Tabulates each students mark and stores that information in a database.
  2. createstatsb - Generates statistics for each question.
  3. mailmarks - Emails to each student the result and a statistical breakdown on their marks. Click here to see a sample email message.
  4. statsguy - Reads the statistical database and creates a report on each question. Click here to see example report.
