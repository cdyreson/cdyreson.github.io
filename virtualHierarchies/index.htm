Virtual Hierarchies

The goal of this project is to create a virtual hierarchy for querying XML data.

The best way to learn about virtual hiearchies is to read our SIGMOD paper. You can download the eXist virtual hierarchies code (eXistVDLN.tar.gz), which is open source, and written in Java. The experimental package for the SIGMOD experiments is located here (sigmod2014.tar.gz), it is 7GB.
