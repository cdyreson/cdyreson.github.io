NAME OODatabase.pm

This package includes all the things required for the Configuration of a simple (non-concurrent) database based on DBM files.

DESCRIPTION

It has several parts

  • OODatabase::Overview, OODatabase::Licence, and OODatabase::Version.

  • OODatabase::Database - The database itself.

  • OODatabase::Table - A table in the database. All tables have two columns.

  • OODatabase::Tuple - A tuple in a table. All tuples have a key and a value.

  • OODatabase::Persistent - An class to implement persistence for an object. All objects stored in a table should inherit this class.
