NAME ExpressionTree::Operations

All the operations in an expression tree.

DESCRIPTION

Below are listed the supported operations.

  • Operations::Comparison - comparison operations

  • Operations::Arithmetic - arithemetic operations

  • Operations::Logical - logical (boolean) operations

  • Operations::Aggregate - aggregate operations

  • Operations::Match - a MATCH operation

  • Operations::Slice - a SLICE operation

  • Operations::Coalesce - a COALESCE operation

  • Operations::Collapse - a COLLAPSE operation

  • Operations::Nodes - a NODES operation

  • Operations::Dimension - a DIMENSION operation

  • Operations::Null - a NULL operation

  • Operations::NonNull - a NONNULL operation
