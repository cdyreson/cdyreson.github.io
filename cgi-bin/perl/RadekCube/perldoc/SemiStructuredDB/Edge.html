NAME Edge

Class encapsulating an Edge in a SemiStructured Database

This package is part of the SemiStructuredDB manpage.

DESCRIPTION

A Edge is a list from, label, and to.

METHODS

new(Id from, Label label, Id to)

  • to

    - The from node, a the Node manpage

  • label

    - The label, a the Label manpage

  • from

    - The to node, a the Node manpage

An Edge is a ordered triplet.

toString()

Returns a string image of an Label.

image()

Return a string image of the Label.

from()

Return the from node.

to()

Return the to node.

label()

Return the label.

slice(Region region)

  • region

    - Slice this region from each label on the edge.

Returns a new edge with the regions in the label that were sliced from the existing edge. Returns 0 if no label could be sliced.

match(Region region)

  • region

    - Match this region against the label on the edge.

Returns a new edge with the regions in the label that match the edge.
