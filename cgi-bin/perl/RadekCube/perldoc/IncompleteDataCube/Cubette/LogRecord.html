NAME LogRecord

Class encapsulating a record in a logfile.

This package is part of IncompleteDataCube::Cubette. For more information on the cube see the <a href=``IncompleteDataCube/Overview.html''>Overview</a>.

DESCRIPTION

A log record is a structure consisting of units above a unit. It is used to update the count associated with filters for each record read from a log file.

METHODS

new($units, $unitToMeasureTables, $coarserUnitGraphs)

  • $units

    - An DBMDatabase::IdList. A list of units to look above in each dimension.

  • $unitToMeasureTables

    - An array of DBMDatabase::Table. Tables that map units to particular measures.

  • $coarserUnitGraphs

    - An array of persistent::PersistentGraph. Directed graphs that lead from finer to coarser units.

Constuct a new log record.

unitsAtMeasures($measures)

  • $measures

    - A <DBMDatabase::IdList> of measures.

Builds a list of the units in each dimension for a measure. Returns the constructed list.
