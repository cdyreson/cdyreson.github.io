NAME IncompleteDataCube::Globals

The global variables in the cube, maintained in a global object. See also IncompleteDataCube::Constants.

DESCRIPTION

Below is the list of instance variables in the global object.

  • dimensions

    - The number of dimensions in the cube.

  • finerUnitGraphs

    - An array of persistent::PersistentGraph. The finer unit graphs map each coarser unit to a set of finer units that are contained by it (but not the transitive closure, just the minimal set of edges). There is one graph for each dimension.

  • coarserUnitGraphs

    - An array of persistent::PersistentGraph. The coarser unit graphs map each finer unit to a set of coarser units that contain it (but not the transitive closure, just the minimal set of edges). There is one graph for each dimension.

  • finerMeasureGraphs

    - An array of persistent::PersistentGraph. The finer measure graphs map each coarser measure to a set of finer measures that are below it in the measure hierarchy (but not the transitive closure, just the minimal set of edges). There is one graph for each dimension.

  • coarserMeasureGraphs

    - An array of persistent::PersistentGraph. The coarser measure graphs map each finer measure to a set of coarser measures that are just above it in the measure hierarchy (but not the transitive closure, just the minimal set of edges). There is one graph for each dimension.

  • unitToMeasureTables

    - An array of database::Table. Each unit to measure table maps a unit to the measure to which it belongs. There is one table for each dimension.

  • measureTables

    - An array of database::Table. Each measure table is a list of measures for that dimension. There is one table for each dimension.

METHODS

new()

Allocate and intialize the necessary variables.

close()

Close and write all the persistent variables are written to disk. This should be called prior to exiting, but even if it not called, in theory, all the persistent variables should be maintained since Perl will clean them up nicely for us by doing the appropriate unties.

save()

Write all the persistent variables are written to disk.
