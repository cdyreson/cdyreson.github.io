NAME Domains.pm

This package includes all the basic domains. See also the OODatabase manpage and the SemiStructuredDB manpage.

DESCRIPTION

The following domains are available.

  • Domains::Boolean - A three-valued boolean.

  • Domains::Integer - Integers.

  • Domains::String - Strings.

  • Domains::TimePoint - A point in time.

  • Domains::CurrentTime - Special routines to figure out the current time.

  • Domains::TimeInterval - An interval in time.

  • Domains::CNF - Conjunctive Normal From stuff, for security domains.

  • Domains::Quality - A quality (low to high).

  • Domains::Dollar - A dallar amount.

  • Domains::Null - A null value.
