NAME SemiStructedDB.pm

This class is the configuration for the SemistructuredDB.

DESCRIPTION

This package has many pieces.

  • SemiStructuredDB::Required - A Required property class.

  • SemiStructuredDB::Region - A region in property space

  • SemiStructuredDB::Label - An edge label

  • SemiStructuredDB::Edge - An edge

  • SemiStructuredDB::Node - A node

  • SemiStructuredDB::EdgeList - An edge list

  • SemiStructuredDB::Path - A sequence of edges

  • SemiStructuredDB::LabelledGraph - A persistent graph class. The graph is the semistructure.

  • SemiStructuredDB::Globals - Global information for the semistructure.

  • SemiStructuredDB::Constants - Constant information for the semistructure.
