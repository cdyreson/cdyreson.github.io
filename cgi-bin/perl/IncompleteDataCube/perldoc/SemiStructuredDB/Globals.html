NAME SemiStructuredDB::Globals

The global state in the semistructured db.

DESCRIPTION

Below is the list of instance variables in the global object.

  • SSGraph

    - A the LabelledGraph manpage. The SSGraph is the raw semistructured graph.

METHODS

new()

Allocate and intialize the necessary variables.

close()

Close and write all the persistent variables are written to disk. This should be called prior to exiting, but even if it not called, in theory, all the persistent variables should be maintained since Perl will clean them up nicely for us by doing the appropriate unties.

save()

Write all the persistent variables are written to disk.
