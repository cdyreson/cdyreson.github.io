NAME QueryCondition

This class represents the satisfaction condition for a query. A query can be partially satisfied.

This package is part of cube::cubette. For more information on the cube see the cube::Overview.

Copyright &copy 1997 Curtis E. Dyreson. All rights reserved. Please be aware of the cube::Licence and cube::Version.

DESCRIPTION

When a query is evaluated, the cubettes are tested to determine if they satisfy the query. This class records how fully the query is satisfied since it may be partially satisfied.

METHODS

new($dimensions, $unitsMatched, $measuresMatched)

  • $dimensions

    - The number of dimensions (yeah, sure it is a global constant too!)

  • $unitsMatched

    - How many units were matched.

  • $measuresMatched

    - How many measures where matched.

Create a query satisfacation object.
