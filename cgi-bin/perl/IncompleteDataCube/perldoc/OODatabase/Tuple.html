DESCRIPTION

A tuple is a pairing of a key and a value.

METHODS

new(Object key, Object value)

  • key

    - The key for the tuple, a OODatabase::Persistent object.

  • value

    - The value for the tuple, a OODatabase::Persistent object.

Create a tuple with the given key, value pair.

newAlreadySwizzled(ByteString key, ByteString value)

  • key

    - The key for the tuple, an already swizzled OODatabase::Persistent object.

  • value

    - The value for the tuple, an already swizzled OODatabase::Persistent object.

Create a tuple with the given key, value pair.

getKey

Retrieve the key.

getValue

Retrieve the value.
