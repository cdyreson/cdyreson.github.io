Current Released Projects

Symbiota2 - A biodiversity database appplicaiton and NSF-sponsored project. Rewrite of Symbiota in Typescript to use web services, Docker, Elasticsearch.

TJKS - This site contains code for temporal hierarchical keyword search. Java and BerkeleyDB.

PolyverseJava - Supporting the Polyverse paradigm for Java with a PolyverseJava to Java translator. ANTLR and Java.

Plug and play SQL - Plug and Play SQL adds a hierarchical query guard to SQL. Written using ANTLR and Java.

AutoRest - AutoREST is a plug-and-play web service architecture. ANTLR and Java.

Old Projects

AUCQL - A prototype semistructured DB. Written in (object-oriented) Perl, with extensive POD.

Incomplete Data Cube - A prototype incomplete data cube. Written in both Perl and Java. The code needs to be updated.

Jumping Spider - A web spider and indexer. Written in Perl, with a fair bit of documentation.

Educational software for the Web - Quizzes, surveys, and self-assessments. These have been battle tested by the students at JCU and Bond for the last six years (thanks everyone).

Fast Calendar I/O - Timegm and gmtime, only faster, in C. These are provably faster, and are 64-bit friendly.

MultiCal - C code to add calendar and time support to databases. A C++ interface may now be available.

Indeterminacy - C code for indeterminate timestamp support.
