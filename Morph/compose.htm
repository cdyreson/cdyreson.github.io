Transformations can be composed using the syntax 
  COMPOSE xform1 WITH xform2

Morphing a morph: 
 
   COMPOSE MORPH publisher [ title [ author [ name ] ] ]
   WITH MORPH author [ name title ]
To run the tutorial queries, use the demo code.

The transformation applies a MORPH to the result of a MORPH. The first MORPH produces a shape which lists the titles published in each year and within each title the authors for that title. The second MORPH chooses authors, and places names and titles as children.

COMPOSE will typically be used with MUTATE or TRANSLATE transformations.
