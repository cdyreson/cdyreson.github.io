The DROP sub-transformation removes the subtree rooted at a label from a shape. It is usually placed within a MUTATE.

Drop titles from the data.

DROP titles from the data: 
MUTATE (DROP title)
To run the tutorial queries, use the demo code.

Want book data only, but remove the titles.

DROP titles from book data, want only book data: 
COMPOSE MORPH book [**] WITH MUTATE (DROP title)
To run the tutorial queries, use the demo code.
