XMorph 2.0

XMorph is an XML data transformation language and implementation. XMorph makes it easy to change the shape on an XML data collection.

The best way to learn about XMorph is to try the tutorial, which has many examples that run in our on-line demo. An brief overview and research papers about XMorph explain some of the technical details in the project. You can download XMorph, which is open source, and written in Java.

We recently combined XMorph with eXist, a native XML DBMS, to support XQuerying of virtual documents. More on this project can be found here.
