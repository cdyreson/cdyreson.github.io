Below we outline the current state of the implementation. While the Jumping Spider is currently functional (please try the interactive demo) much remains to be done.

Implemented features

  • Robot to collect information.
  • Graph routines.
  • Jumping spider index.

Unimplemented features (wish list)

  • Enriched query language.

Dependencies (available from CPAN)

  • LWP package.
