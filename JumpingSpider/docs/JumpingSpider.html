NAME JumpingSpider.pm

This package includes all the things required for the Jumping Spider. For more see JumpingSpider::Overview and JumpingSpider::Licence.

DESCRIPTION

Included are several parts of the Jumping Spider.

  • JumpingSpider::Query - Utilities useful in Querying.

  • JumpingSpider::WWWGraph - Utilities useful in building the WWWGraph.

  • JumpingSpider::Globals - All the database tables needed.

  • the DBMDatabase manpage - The DBMDatabase, a simple kind of database based on DBM files.

  • Persistent::PersistentGraph - A persistent graph.

  • JumpingSpider::Constants - All the constants needed.

  • JumpingSpider::TitleTable - Useful title table utilities.
