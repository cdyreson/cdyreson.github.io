Jumping Spider/Mini-World Web : Implementation

  • POD Documenation - on-line documentation for the JumpingSpider Perl code pacakage
  • version 1.00 (August 6, 1998) Perl code (tarred, gzipped perl modules) (Perl 5)

Below we outline the current state of the implementation. While the Jumping Spider/Mini-World Web is currently functional (please try the interactive demo) much remains to be done.

Implemented features

  • Robot to collect information.
  • Graph routines.
  • Jumping spider index.

Unimplemented features (wish list)

  • Enriched query language.

Dependencies (available from CPAN)

  • LWP package.
