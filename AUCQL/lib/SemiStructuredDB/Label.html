NAME Label

Class encapsulating an Edge Label in a SemiStructured Database

This package is part of the SemiStructuredDB manpage.

DESCRIPTION

A Label is a list of property space regions (see the Region manpage). The label supports matching, slicing, coalescing, and collapsing operations.

METHODS

new(List of Regions r)

  • RegionList r

    - The list of the Region manpages

A Label is a list of property space the Region manpages. This creates a Label from a list of the Region manpages.

toString()

Returns a string image of an Label.

image()

Return a string image of the Label.

regions()

Return a reference to the list of regions in a Label.

insert(Region r)

  • RegionList r

    - The list of the Region manpages

Insert a region into the list.

dimensionExtract(string dimension)

  • string dimension

    - The name of the dimension to extract

Extracts the dimension value from each region in the list building a new list.

slice(Region S)

  • Region S

    - The region to slice with.

Slices a portion from each region in the label, creating a new label.

collapse(Label other)

  • Label other

    - The label to collapse with.

Collapses this label with another, creating a new label. The collapsing is the pairwise collapsing of each region in each label. Typically, this will be one region in, one region out.

This function will fail (return value of undef) if there is no collapsed label (the resulting label has some empty property in all Regions).

match(Region r)

  • Region r

    - The region to match against.

Matches a region to every region in this label, returns a new Label that contains all the matching regions, or 0 (false) if no region matches.

coalesce(String propertyName)

  • String propertyName

    - The propertyName to coalesce on.

Coalesce all the Regions in this Label on the indicated property.
