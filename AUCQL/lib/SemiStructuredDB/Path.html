NAME Path

Class representing a path value. A path is a sequence of SemiStructuredDB::Edges.

This package is part of the SemiStructuredDB manpage.

DESCRIPTION

A Path is a sequence of the Edge manpages.

METHODS

new(Edge edge)

  • edge

    - A SemiStructuredDB::Edge to start the path with, this edge is optional.

Construct an empty path, or a path with one edge.

collapse()

Fetch the collapsed path.

dimensionExtract(string dimension)

Fetch the dimension value from the collapsed area. Returns a list of values.

clone()

Clone the path.

append(Edge edge)

  • edge

    - The SemiStructuredDB::Edge to append.

Adds an SemiStructuredDB::Edge to the path.

prepend(Edge edge)

  • edge

    - The SemiStructuredDB::Edge to prepend.

Prepends a SemiStructuredDB::Edge to the start of a path.

terminalNode()

Return the final node in the path.

startNode()

Return the starting node in the path.

image()

Create a formatted string of all the edges in the path

slice(Region region)

  • region

    - The SemiStructuredDB::Region to slice with.

Slice every edge in the path using the region. Return 0 if no valid path results. Return a new sliced path if a valid sliced path exists. Note that this will leave the path unchanged, it generates a new path (this method is not a mutator).

resultType()

Result type is path.

getValue()

The value of the path is the value of the final node in the path.

presentation()

Present this path as the final node in the path.
