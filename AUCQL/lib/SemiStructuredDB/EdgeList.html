NAME EdgeList

Class encapsulating a set of SemiStructuredDB::Edges.

This package is part of the SemiStructuredDB manpage.

DESCRIPTION

An EdgeList is a data type for columns in a database the Table manpage. An EdgeList cannot be used as a key.

METHODS

new(Node from, list a)

  • from

    - The reference to a the from node object.

  • a

    - The reference to a list of edges to store, often it is an empty list.

Construct an empty List

insert(Edge edge)

  • edge

    - The SemiStructuredDB::Edge to insert.

Inserts an SemiStructuredDB::Edge into this list.
