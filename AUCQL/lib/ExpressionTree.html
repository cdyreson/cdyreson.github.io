NAME ExpressionTree.pm

A class for an expression tree.

DESCRIPTION

An expression tree is an important thing.

  • ExpressionTree::Variable - a variable node

  • ExpressionTree::Operand - an operand in an operation

  • ExpressionTree::Value - an extract value operation

  • ExpressionTree::Operations - expression tree operations

  • ExpressionTree::RegExp - regular expressions
