NAME ExpressionTree::RegExp::Token

DESCRIPTION

This class represents matching of a single 'token' in a regular expression, to the current place in a path.

METHODS

new(Region token)

  • token

    - The the Region manpage to match.

Create a new Token operation.

resultType()

Return the type of result.

dump($indent)

Dump this node.

evaluate(LabelledGraph graph, Path path)

  • graph

    - A SemiStructuredDB::LabelledGraph that stores the semistructure.

  • path

    - A SemiStructuredDB::PPath, the current path in the graph.

Compare the token against all the edges out of the terminal node in the current path.
