NAME Domains.pm

This package includes all the basic domains. See also the OODatabase manpage and the SemiStructuredDB manpage.

DESCRIPTION

The following domains are available.

  • the Domains::Boolean manpage - A three-valued boolean.

  • the Domains::Integer manpage - Integers.

  • the Domains::String manpage - Strings.

  • the Domains::TimePoint manpage - A point in time.

  • the Domains::CurrentTime manpage - Special routines to figure out the current time.

  • the Domains::TimeInterval manpage - An interval in time.

  • the Domains::CNF manpage - Conjunctive Normal From stuff, for security domains.

  • the Domains::Quality manpage - A quality (low to high).

  • the Domains::Dollar manpage - A dallar amount.

  • the Domains::Null manpage - A null value.
