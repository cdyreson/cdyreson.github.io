NAME SemiStructedDB.pm

This class is the configuration for the SemistructuredDB.

DESCRIPTION

This package has many pieces.

  • the SemiStructuredDB::Required manpage - A Required property class.

  • the SemiStructuredDB::Region manpage - A region in property space

  • the SemiStructuredDB::Label manpage - An edge label

  • the SemiStructuredDB::Edge manpage - An edge

  • the SemiStructuredDB::Node manpage - A node

  • the SemiStructuredDB::EdgeList manpage - An edge list

  • the SemiStructuredDB::Path manpage - A sequence of edges

  • the SemiStructuredDB::LabelledGraph manpage - A persistent graph class. The graph is the semistructure.

  • the SemiStructuredDB::Globals manpage - Global information for the semistructure.

  • the SemiStructuredDB::Constants manpage - Constant information for the semistructure.
