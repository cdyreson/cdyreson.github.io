NAME ExpressionTree::Operations

All the operations in an expression tree.

DESCRIPTION

Below are listed the supported operations.

  • the Operations::Comparison manpage - comparison operations

  • the Operations::Arithmetic manpage - arithemetic operations

  • the Operations::Logical manpage - logical (boolean) operations

  • the Operations::Aggregate manpage - aggregate operations

  • the Operations::Match manpage - a MATCH operation

  • the Operations::Slice manpage - a SLICE operation

  • the Operations::Coalesce manpage - a COALESCE operation

  • the Operations::Collapse manpage - a COLLAPSE operation

  • the Operations::Nodes manpage - a NODES operation

  • the Operations::Dimension manpage - a DIMENSION operation

  • the Operations::Null manpage - a NULL operation

  • the Operations::NonNull manpage - a NONNULL operation
