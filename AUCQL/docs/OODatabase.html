NAME OODatabase.pm

This package includes all the things required for the Configuration of a simple (non-concurrent) database based on DBM files.

DESCRIPTION

It has several parts

  • the OODatabase::Overview manpage, the OODatabase::Licence manpage, and the OODatabase::Version manpage.

  • the OODatabase::Database manpage - The database itself.

  • the OODatabase::Table manpage - A table in the database. All tables have two columns.

  • the OODatabase::Tuple manpage - A tuple in a table. All tuples have a key and a value.

  • the OODatabase::Persistent manpage - An class to implement persistence for an object. All objects stored in a table should inherit this class.
