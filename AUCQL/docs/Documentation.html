NAME Documentation

This package includes all the documentation associated with the project.

DESCRIPTION

It has many parts.

  • the Documentation::Overview manpage - An overview of the project.

  • the Documentation::Licence manpage - The licence for the code.

  • the Documentation::Version manpage - The version number for this release.

  • The the loadDB manpage command - Evaluate a sequence of INSERT statements

  • The the evaluateQuery manpage command - Evaluate a SELECT statement (and default SETs).

  • the SemiStructuredDB manpage - The semistructured database classes.

  • the Dimensions manpage - Which dimensions are available.

  • the OODatabase manpage - A DBM based OO database. Used to store the persistent objects.

  • the ErrorHandler manpage - The error handler

  • the ExpressionTree manpage - The expression tree

  • the Evaluator::Select manpage - The query evaluation engine for the select statement

  • the Domains manpage - The domains of labels.
