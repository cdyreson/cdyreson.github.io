NAME IncompleteDataCube.pm

This class is the configuration for the entire incomplete data cube.

DESCRIPTION

This package needs two other pieces.

  • the DBMDatabase manpage - A DBM based database. Used to store the cube information.

  • Persistent::Graph - A persistent graph class. The graph maintains information about the hierarchy of units and measures in the cube.

This package has several parts.

  • IncompleteDataCube::Cubette - A cubette is a complete sub-cube within the incomplete data cube.

  • IncompleteDataCube::Globals - Global constants to configure the cube.

  • IncompleteDataCube::Constants - The global constants, EDIT THIS TO CONFIGURE THE CUBE.

  • IncompleteDataCube::Dimension - A class to create the global variables for a new dimension, a helper class for IncompleteDataCube::Globals.

For more information on the cube see the IncompleteDataCube::Overview. Please be aware of the IncompleteDataCube::Licence and IncompleteDataCube::Version.
