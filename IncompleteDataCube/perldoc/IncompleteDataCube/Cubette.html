NAME IncompleteDataCube::Cubette.pm

This class implements a complete sub-cube within the cube.

DESCRIPTION

It has several parts

  • Cubette::Cubette - The top-level interface to a complete sub-cube within the incomplete data cube.

  • Cubette::CubetteSpecification - A Cubette Specification is a list of units and measure.

  • Cubette::LogRecord - A class to iterate over a log file and count instances of particular records that are filtered through the cubette specifications.

  • Cubette::QueryCondition - A class that represents the degree of satisfaction for a query.

For more information on the cube see the <a href=``IncompleteDataCube/Overview.html''>Overview</a>.
