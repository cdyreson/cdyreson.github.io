NAME IncompleteDataCube::Constants

EDIT THIS TO CONFIGURE THE CUBE. The global constants in the cube.

For more information on the cube see the <a href=``IncompleteDataCube/Overview.html''>Overview</a>.

DESCRIPTION

A list of global available constants.

  • dimensions

    - The number of dimensions in the cube.

  • dimensionNames

    - An array of strings, the names of the dimensions.

  • logFileSplitFileName

    - The name of the file that contains the flex specifications to split the log file by dimension. It is assumed to be relative to the readspec subdirectory.

  • filterFileName

    - The name of the file that contains the filter specifications. It is assumed to be relative to the readspec subdirectory.

  • databaseName

    - In what directory will the database created by the cube live?

  • stringTableName

    - The table containing strings used by database::Convert.

  • idTableName

    - The table containing Ids used by database::Convert.

  • finerUnitGraphName

    - The finer unit graphs.

  • coarserUnitGraphName

    - The coarser unit graphs.

  • finerMeasureGraphName

    - The finer measure graph for each dimension.

  • coarserMeasureGraphName

    - The coarser measure graph for each dimension.

  • unitToMeasureTableName

    - The table that maps units to measures.

  • measureTableName

    - The table that lists the available measures in each dimension.

  • filterTableName

    - The table that lists the filters.

  • filterUnitTableName

    - The table of filter units.

  • filterMeasureTableName

    - The table of filter measures.

  • countTableName

    - The table that contains counts for each filter.
