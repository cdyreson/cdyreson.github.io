NAME Cubette

The top-level interface to a complete sub-cube within the incomplete data cube.

This package is part of IncompleteDataCube::Cubette. For more information on the cube see the <a href=``IncompleteDataCube/Overview.html''>Overview</a>.

Copyright &copy 1997 Curtis E. Dyreson. All rights reserved. Please be aware of the <a href=``IncompleteDataCube/License.html''>license</a> and <a href=``IncompleteDataCube/License.html''>version</a>.

DESCRIPTION

A Cubette is a complete sub-cube. This class advertises operations that are available on Cubettes.

METHODS

query($cubetteSpec, $queryUnitsAbove, $queryMeasuresBelow)

  • $cubetteSpec

    - An IncompleteDataCube::Cubette::CubetteSpecification that specifies the units and measures in the query.

  • $queryUnitsAbove

    - An array of DBMDatabase::IdSet. Each DBMDatabase::IdSet is the set of units above the query's unit in the respective domain in the query.

  • $cubetteMeasuresBelow

    - An array of DBMDatabase::IdSet. Each DBMDatabase::IdSet is the set of measures below the query's measure in the respective domain in the query.

Determine if a query is satisfied by the specified Cubette. Return true if the query can be satisfied, false otherwise. For more on satisfaction, see the relevant papers (e.g., the VLDB paper).

History

  • Nov 14, 1997

    - Added count of unit and measure matching. It now returns the number of units and measures matched as a list.

relaxQuery($cubetteSpec, $queryUnitsAbove, $queryMeasuresBelow)

  • $cubetteSpec

    - An IncompleteDataCube::Cubette::CubetteSpecification that specifies the units and measures in the query.

  • $queryUnitsAbove

    - An array of DBMDatabase::IdSet. Each DBMDatabase::IdSet is the set of units above the query's unit in the respective domain in the query.

  • $cubetteMeasuresBelow

    - An array of DBMDatabase::IdSet. Each DBMDatabase::IdSet is the set of measures below the query's measure in the respective domain in the query.

Determine if a query can be relaxed to be satisfied by the specified Cubette. Return true if the query can be relaxed, false otherwise. For more on query relaxation or generalization, see the relevant papers (e.g., the VLDB paper).

sum($satisfyingCubette, $query, $finerUnitGraphs, $unitToMeasureTables, $countTable)

  • $satisfyingCubette

    - An IncompleteDataCube::Cubette::CubetteSpecification of the cubette that we know satisfies a query.

  • $query

    - An IncompleteDataCube::Cubette::CubetteSpecification of the query.

  • $finerUnitGraphs

    - An array of persistent Persistent::Graphs. Each finer unit graph is a graph of which sub-units that are within a given unit (edges go from coarser to finer units).

  • $unitToMeasureTables

    - An array of database DBMDatabase::Tables. Each unit to measure table maps a unit to the corresponding measure.

  • $countTable

    - A DBMDatabase::Table. The count for each cubette.

Compute a sum aggregate for the answer. Only call this after you have assured that the cubette is satisfying! For now we will assume a strict hierarchy.

increment($cubette, $log, $finerUnitGraphs, $unitToMeasureTables, $countTable, $doneThisRound, $verbose)

  • $cubette

    - An IncompleteDataCube::Cubette::CubetteSpecification of the cubette to increment.

  • $log

    - An IncompleteDataCube::Cubette::CubetteSpecification of the log file?

  • $filterUnitTable

    - A DBMDatabase::Table. The units for each filter.

  • $filterMeasureTable

    - A DBMDatabase::Table. The measures for each filter.

  • $countTable

    - A DBMDatabase::Table. The count for each cubette.

  • $doneThisRound

    - An DBMDatabase::IdSet. A set of filters that have been counted.

  • $verbose

    - Turn on if verbose reporting wanted.

Increment the count associated with this filter.

quickIncrement($countList, $countTable)

  • $countList

    - A DBMDatabase::IdList of the units to increment the count of.

  • $countTable

    - A DBMDatabase::Table. The count for each cubette.

Quickly increment the count for a particular multidimensonal unit.
