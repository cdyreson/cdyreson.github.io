NAME CubetteSpecification

The CubetteSpecification class implements the specification of a cubette in terms of units and measures.

This package is part of IncompleteDataCube::Cubette. For more information on the cube see the <a href=``IncompleteDataCube/Overview.html''>Overview</a>.

METHODS

new($cubette, $filterUnitTable, $filterMeasureTable)

  • $cubette

    - A DBMDatabase::Id? The cubette.

  • $filterUnitTable

    - A DBMDatabase::Table. The units for each filter.

  • $filterMeasureTable

    - A DBMDatabase::Table. The measures for each filter.

Construct a specification.

image()

Create a nice formatted image of the Cubette.

imageWithMeasures()

Create a nice formatted image of the Cubette with measures, e.g, 

   World@continents, 1997@months, Incomplete Cube@Curtis Dyreson

getUnits()

Return an DBMDatabase::IdList of the units in this cubette.

getMeasures()

Return an DBMDatabase::IdList of the measures in this cubette.
